# Selenium-cucumber-OrangeHrm

### Technology:

* Tool: Selenium cucumber
* IDE: Intellij
* Build tool: Maven
* Language: Java
* Test_Runner: JUnit

### Run the Automation Script by the following command:

```bash
  mvn clean test 
```
